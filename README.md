# 🏗️ Gestion Chantier MHAMID v3.0

Application de gestion de chantier Fibre & RTC avec système d'alertes email et analytics avancés.

![Version](https://img.shields.io/badge/version-3.0.0-blue)
![Streamlit](https://img.shields.io/badge/streamlit-1.28+-red)
![Python](https://img.shields.io/badge/python-3.8+-green)

## 📋 **Fonctionnalités**

- ✅ Gestion des instances avec motifs
- ⚠️ Suivi des dérangements
- ⚖️ Gestion des litiges
- 🔧 Programme de fiabilisation
- 📧 Système d'alertes email automatiques
- 📊 Dashboard avec analytics avancés
- 📥 Import/Export Excel multi-sheets
- 📈 Statistiques avancées (heatmaps, tendances)

## 🚀 **Installation**

### Prérequis

- Python 3.8 ou supérieur
- pip (gestionnaire de paquets Python)

### Installation des dépendances

```bash
# Cloner le repository
git clone https://github.com/blhomerenov-arch/bl_service.git
cd bl_service

# Installer les dépendances
pip install -r requirements.txt
